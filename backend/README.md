## Prerequisites
Make sure you have Node JS installed
```
https://nodejs.org/en/download
```

## How to run Servers
There are two server configurations you can run. One built with Fastify Framework the other built with Express JS

### Express JS
```bash
npm run start
```

### Fastify
```bash
npm run fast
```
